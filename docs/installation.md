# Installation

## Homebrew

```bash
brew install ionrock/tap/we
```

## Go install

```bash
go install github.com/ionrock/we/cmd/we@latest
```

## Binary releases

Download a release from <https://github.com/ionrock/we/releases> and place the `we` binary somewhere on your `PATH`.

## Verify

```bash
we --version
we --help
```
